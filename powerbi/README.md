## Power BI Folder

This folder contains Power BI files used to build the interactive churn analysis dashboard.

### Dashboard Highlights
- Key KPIs: Total Customers, Churned Customers, Churn Rate, Average Tenure
- Churn trend over customer tenure
- Contract type and payment method churn analysis
- Tenure group based churn distribution
- Monthly charges vs tenure scatter analysis

### Notes
- Screenshots and visuals are provided in the `visuals/` folder for reference.
- The dashboard uses a custom dark-mode theme for clarity and consistency.
