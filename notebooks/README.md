## Notebooks Folder

This folder contains Jupyter notebooks used for data preprocessing and exploratory data analysis (EDA).

### Contents
- Data loading and inspection
- Data cleaning and type conversion
- Feature engineering (churn flag, tenure groups)
- Exploratory analysis of churn patterns and customer behavior

### Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn

The cleaned dataset generated here was exported and used for Power BI visualization.
